Crime-Park Correlation Dashboard

Crime & Park Dashboard

Interactive · D3.js + Flask

An interactive dashboard exploring district-level patterns and correlations between crime, park area, and GDP in Seoul (2015–2022).

This dashboard uses 2015–2022 district-level data for Seoul to explore how crime, park area, and GDP (GRDP) vary across gu. Built with a Flask backend and D3.js + DC.js + Crossfilter on the frontend, it supports interactive filtering and linked updates across the map and charts.

  • ✦ Time span: 2015–2022
  • ✦ Spatial unit: Districts (gu), Seoul
  • ✦ Metrics: Crime rate, Park area, GDP (GRDP)
  • ✦ How to use: Click metric buttons to toggle the map view. Select one metric to see its distribution by district. Select two metrics to switch into correlation mode: Crime × Park or Crime × GDP. Click again to deselect.
  • ✦ Interaction: Use the legend to interpret colors; click a district to filter/focus. Use the year chart to filter the time range.

Interpretation note: Correlation reflects co-variation, not causation. Patterns may be influenced by unobserved factors (e.g., population, reporting practices, land use, development).

Color Diagram

Color Diagram

Interactive · NLP

An NLP-powered platform that interprets written language and suggests matching color palettes.

Explore the relationship between text and color — This project introduces a text-based color recommendation platform built on natural language processing (NLP). By analyzing written input such as words, phrases, or short descriptions, the system recommends color palettes that reflect both semantic meaning and emotional tone.

  • ✦ Focus: Mapping language to perceptual color choices
  • ✦ Tools: NLP (transformer-based models), Semantic Analysis, React for visualization
  • ✦ Effect: Reveals how linguistic features (mood, metaphor, affect) can be expressed through color
  • ✦ Interaction: Type any text to generate adaptive color palettes
